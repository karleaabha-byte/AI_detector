# 👾 AI vs Real Image Detector

A deep learning-based web app that detects whether an image is **AI-generated or real** using a CNN model trained on CIFAKE dataset.

---

## 🚀 Features

- 🧠 CNN-based classification (TensorFlow/Keras)
- 🖼 Image upload & real-time prediction
- 📊 Performance dashboard
- 📉 ROC Curve & AUC
- 📋 Classification report
- 📐 Statistical analysis:
  - Maximum Likelihood Estimation (MLE)
  - Hypothesis Testing
  - Misclassification comparison

---

## 🧪 Dataset

- CIFAKE Dataset (Kaggle)
- Classes:
  - REAL images
  - AI-generated images
- Image size: 32×32

---

## 🛠 Tech Stack

- Python
- Streamlit
- TensorFlow / Keras
- NumPy, Pandas
- Matplotlib, Seaborn
- Scikit-learn
